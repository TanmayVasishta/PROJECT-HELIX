================================================================================
WHAT HELIX IS
================================================================================

HELIX is a privacy-first Artificial Intelligent Operating System (AIOS)
middleware layer. It is a desktop assistant that automates local OS workflows
(file management, directory organization, app launching, workflow automation)
using a voice/text interface. It resolves the conflict between data sovereignty
and advanced AI reasoning using a Hybrid Edge-Cloud Architecture:

  - The Sentinel Node (local Llama3/Qwen via Ollama) handles OS tasks and
    mathematically redacts PII before anything leaves the machine.
  - The Cloud Oracle (Google Gemini) handles complex reasoning, but ONLY
    receives sanitized, PII-free prompts screened by the Sentinel.
  - ChromaDB provides persistent local vector memory (RAG) across sessions.
  - PyQt6 provides a non-blocking async Voice HUD (Phase 5).
  - Watchdog provides proactive filesystem monitoring.
